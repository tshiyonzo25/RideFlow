# RideFlow Backend

A Django REST API for a ride booking system.

## Features

- Customer management
- Driver management
- Booking system
- RESTful APIs

## Tech Stack

- Python 3.13
- Django
- Django REST Framework

## Setup Instructions

```bash
git clone https://github.com/tshiyonzo25/RideFlow.git
cd RideFlow
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
